Views & Indexes

Views (Vistas)

Una vista es una tabla virtual basada en el conjunto de resultados de una consulta SQL. Contiene filas y columnas, igual que una tabla real.

Crear una Vista

CREATE VIEW usuarios_activos AS
SELECT id, nombre, email
FROM usuarios
WHERE activo = 1;

Usar una Vista

SELECT * FROM usuarios_activos;

Indexes (Índices)

Los índices se utilizan para recuperar datos de la base de datos de manera más rápida. Los usuarios no pueden ver los índices, solo se usan para acelerar las búsquedas/consultas.

Crear un Índice

CREATE INDEX idx_email
ON usuarios (email);
Nota: Actualizar una tabla con índices lleva más tiempo que actualizar una tabla sin ellos (porque los índices también deben actualizarse). Por lo tanto, crea índices solo en columnas que se usarán frecuentemente para buscar.
