Ordenamiento y Límites

ORDER BY

La cláusula ORDER BY se utiliza para ordenar el conjunto de resultados en orden ascendente o descendente.

  • ASC: Ascendente (por defecto).
  • DESC: Descendente.

Ejemplo

-- Ordenar usuarios por nombre alfabéticamente
SELECT * FROM usuarios
ORDER BY nombre ASC;
-- Ordenar productos por precio, del más caro al más barato
SELECT * FROM productos
ORDER BY precio DESC;

LIMIT

La cláusula LIMIT se utiliza para especificar el número de registros a devolver. Es muy útil para paginación o para obtener los "top X" resultados.

Ejemplo

-- Obtener los 5 productos más caros
SELECT * FROM productos
ORDER BY precio DESC
LIMIT 5;
