Triggers (Disparadores)

¿Qué es un Trigger?

Un trigger es un conjunto de instrucciones SQL que se ejecutan automáticamente ("se disparan") en respuesta a ciertos eventos en una tabla en particular, como INSERT, UPDATE o DELETE.

Sintaxis Básica

CREATE TRIGGER nombre_trigger
{BEFORE | AFTER} {INSERT | UPDATE | DELETE}
ON nombre_tabla FOR EACH ROW
BEGIN
    -- Código a ejecutar
END;

Ejemplo: Auditoría

Guardar un registro cada vez que se actualiza un salario.

CREATE TRIGGER log_salario_update
AFTER UPDATE ON empleados
FOR EACH ROW
BEGIN
    IF OLD.salario <> NEW.salario THEN
        INSERT INTO auditoria_salarios (empleado_id, salario_anterior, salario_nuevo, fecha)
        VALUES (OLD.id, OLD.salario, NEW.salario, NOW());
    END IF;
END;
