Update: Actualizar Datos

La sentencia UPDATE

La operación Update se utiliza para modificar registros existentes en una tabla.

Sintaxis Básica

UPDATE nombre_tabla
SET columna1 = valor1, columna2 = valor2, ...
WHERE condicion;

Ejemplo

Para actualizar el email de un usuario específico:

UPDATE usuarios
SET email = 'nuevo_email@example.com'
WHERE nombre = 'Juan Perez';
¡Cuidado! Si omites la cláusula WHERE, se actualizarán TODOS los registros de la tabla. 
-- ¡Esto cambiará el email de TODOS los usuarios!
UPDATE usuarios
SET email = 'error@example.com';

Buenas Prácticas

  • Siempre prueba tu cláusula WHERE con un SELECT antes de ejecutar el UPDATE para asegurarte de que estás seleccionando las filas correctas.
  • Haz una copia de seguridad antes de actualizaciones masivas.

Errores Comunes

  • Olvidar el WHERE y actualizar toda la tabla.
  • Actualizar la clave primaria (generalmente no se recomienda).
