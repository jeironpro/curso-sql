Funciones de Agregación

Funciones Comunes

SQL proporciona funciones para realizar cálculos sobre un conjunto de valores y devolver un único valor.

Función Descripción
COUNT() Cuenta el número de filas.
SUM() Suma los valores de una columna numérica.
AVG() Calcula el promedio de una columna numérica.
MAX() Devuelve el valor máximo.
MIN() Devuelve el valor mínimo.

Ejemplos

-- Contar total de usuarios
SELECT COUNT(*) FROM usuarios;
-- Precio promedio de productos
SELECT AVG(precio) FROM productos;

GROUP BY

La sentencia GROUP BY agrupa filas que tienen los mismos valores en filas de resumen.

Ejemplo

-- Contar usuarios por país
SELECT pais, COUNT(*) as total_usuarios
FROM usuarios
GROUP BY pais;

HAVING

La cláusula HAVING se usa para filtrar grupos (ya que WHERE no funciona con funciones de agregación).

-- Países con más de 100 usuarios
SELECT pais, COUNT(*) as total_usuarios
FROM usuarios
GROUP BY pais
HAVING COUNT(*) > 100;
