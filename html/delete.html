Delete: Eliminar Datos

La sentencia DELETE

La operación Delete se utiliza para eliminar registros de una tabla.

Sintaxis Básica

DELETE FROM nombre_tabla
WHERE condicion;

Ejemplo

Para eliminar un usuario específico:

DELETE FROM usuarios
WHERE nombre = 'Juan Perez';
¡Peligro! Al igual que con UPDATE, si omites la cláusula WHERE, se eliminarán TODOS los registros de la tabla. 
-- ¡Esto borrará TODOS los usuarios!
DELETE FROM usuarios;

Buenas Prácticas

  • Al igual que con UPDATE, verifica tu WHERE con un SELECT primero.
  • Considera usar "Soft Delete" (marcar como inactivo en lugar de borrar físicamente) si necesitas mantener un historial.

Errores Comunes

  • Olvidar el WHERE y borrar toda la tabla.
  • Borrar registros que tienen dependencias (claves foráneas) sin configurar ON DELETE CASCADE, lo que causará un error.
