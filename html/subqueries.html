Subqueries (Subconsultas)

¿Qué es una Subquery?

Una subconsulta es una consulta anidada dentro de otra consulta SQL. Se puede usar en cláusulas SELECT, FROM, WHERE, etc.

Subconsulta en WHERE

Usada para filtrar resultados basados en el resultado de otra consulta.

-- Usuarios que han hecho pedidos
SELECT * FROM usuarios
WHERE id IN (SELECT cliente_id FROM pedidos);

Subconsulta en SELECT

Usada para calcular un valor para cada fila.

-- Usuarios y su número de pedidos
SELECT nombre, 
    (SELECT COUNT(*) FROM pedidos WHERE pedidos.cliente_id = usuarios.id) as total_pedidos
FROM usuarios;

Subconsulta en FROM

Usada como una tabla temporal.

-- Promedio de pedidos por usuario
SELECT AVG(total_pedidos)
FROM (
    SELECT cliente_id, COUNT(*) as total_pedidos
    FROM pedidos
    GROUP BY cliente_id
) as pedidos_por_usuario;
