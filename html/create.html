Create: Insertar Datos

La sentencia INSERT

La operación Create en SQL se realiza principalmente mediante la sentencia INSERT INTO. Esta sentencia permite agregar nuevas filas a una tabla.

Sintaxis Básica

INSERT INTO nombre_tabla (columna1, columna2, columna3, ...)
VALUES (valor1, valor2, valor3, ...);

Ejemplo

Supongamos que tenemos una tabla llamada usuarios con las columnas nombre y email.

INSERT INTO usuarios (nombre, email)
VALUES ('Juan Perez', 'juan@example.com');

También es posible insertar múltiples filas a la vez:

INSERT INTO usuarios (nombre, email)
VALUES 
('Maria Garcia', 'maria@example.com'),
('Carlos Lopez', 'carlos@example.com');

Buenas Prácticas

  • Especifica siempre las columnas en el INSERT. Aunque es opcional si insertas valores para todas las columnas, es más seguro y legible especificarlas.
  • Usa transacciones para inserciones masivas críticas.

Errores Comunes

  • Intentar insertar un duplicado en una columna con restricción UNIQUE o PRIMARY KEY.
  • No coincidir el número de valores con el número de columnas especificadas.
  • Intentar insertar NULL en una columna definida como NOT NULL.
