Procedimientos Almacenados

¿Qué es un Stored Procedure?

Un procedimiento almacenado es un código SQL preparado que puedes guardar, de modo que el código se pueda reutilizar una y otra vez.

Crear un Procedimiento

DELIMITER //

CREATE PROCEDURE ObtenerClientesPorPais(IN nombrePais VARCHAR(255))
BEGIN
    SELECT * FROM clientes WHERE pais = nombrePais;
END //

DELIMITER ;

Llamar a un Procedimiento

CALL ObtenerClientesPorPais('España');

Funciones Definidas por el Usuario (UDF)

Similares a los procedimientos, pero devuelven un valor y se pueden usar en sentencias SQL.

CREATE FUNCTION CalcularImpuesto(precio DECIMAL(10,2))
RETURNS DECIMAL(10,2)
BEGIN
    RETURN precio * 0.21;
END;
SELECT nombre, precio, CalcularImpuesto(precio) as iva
FROM productos;
