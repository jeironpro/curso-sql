La sentencia SELECT

La operación Read en SQL se realiza mediante la sentencia SELECT. Esta es probablemente la sentencia más utilizada en SQL.

Sintaxis Básica

SELECT columna1, columna2, ...
FROM nombre_tabla;

Ejemplos

Para seleccionar todas las columnas de la tabla usuarios:

SELECT * FROM usuarios;

Para seleccionar solo el nombre y el email:

SELECT nombre, email FROM usuarios;

Filtrar Datos (WHERE)

Podemos filtrar los resultados usando la cláusula WHERE:

SELECT * FROM usuarios
WHERE nombre = 'Juan Perez';

Buenas Prácticas

  • Evita usar SELECT * en producción. Selecciona solo las columnas que necesitas para mejorar el rendimiento y reducir el tráfico de red.
  • Usa alias (AS) para hacer los nombres de columnas más legibles, especialmente en reportes.

Errores Comunes

  • Olvidar la cláusula WHERE y traer millones de registros innecesariamente.
  • Confundir el orden de las cláusulas (ej. poner ORDER BY antes de WHERE).
