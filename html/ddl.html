DDL: Definición de Datos

Data Definition Language (DDL)

El DDL se encarga de definir la estructura de la base de datos (tablas, esquemas, etc.).

CREATE TABLE

Crea una nueva tabla.

CREATE TABLE usuarios (
    id INT PRIMARY KEY,
    nombre VARCHAR(100),
    email VARCHAR(100),
    fecha_registro DATE
);

ALTER TABLE

Modifica una tabla existente (añadir columnas, cambiar tipos, etc.).

-- Añadir una columna
ALTER TABLE usuarios
ADD telefono VARCHAR(20);

DROP TABLE

Elimina una tabla y todos sus datos.

¡Peligro! Esta acción no se puede deshacer. 
DROP TABLE usuarios;

Tipos de Datos Comunes

  • INT: Números enteros.
  • VARCHAR(n): Texto de longitud variable (hasta n caracteres).
  • DATE: Fecha (YYYY-MM-DD).
  • DECIMAL(p,s): Números decimales exactos.
